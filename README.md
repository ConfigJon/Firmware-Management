# Firmware-Management

PowerShell scripts for managing BIOS/firmware settings and passwords on Dell, HP, and Lenovo systems. Runs on Windows PowerShell 5.1 and PowerShell 7.

For all documentation, see my blog at https://www.configjon.com/.

If you want the classic scripts, see [v2 release overview](https://www.configjon.com/bios-management-scripts-v2/).

If you want the Intune scripts, see [Managing BIOS Passwords and Settings with Intune](https://www.configjon.com/bios-management-with-intune/).
